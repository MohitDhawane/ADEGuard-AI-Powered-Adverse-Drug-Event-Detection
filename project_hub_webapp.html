Project Overview

ADEGuard is an end-to-end AI system designed to detect Adverse Drug Events (ADEs) from unstructured VAERS reports. It transforms noisy, free-text symptom notes into clean, structured, and actionable insights to enhance drug safety monitoring. The core of the system is a fine-tuned BioBERT model that achieves over a **97% F1-Score** in identifying `ADE` and `DRUG` entities.

Setup & Usage

Follow these steps to run the entire project on your local machine.

1. Clone the Repository

git clone [repository-url]

2. Install Dependencies

pip install -r requirements.txt

3. Run the Full Pipeline

Execute the scripts in order to process data and train the model.

4. Launch the Application

Start the backend API first, then the frontend UI in separate terminals.

# Terminal 1: Start Backend
uvicorn api_backend:app --reload

# Terminal 2: Start Frontend
streamlit run streamlit_app.py
